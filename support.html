Support CruxCoach

CruxCoach is built and maintained by an indie developer in their spare time. The app is, and will stay, free and ad-free. If it helps your training, here's how you can keep it going.

Bitcoin Lightning

Easiest, instant, no account needed at the receiving end. Scan the QR with any Lightning-capable wallet — or copy the address into Zeus, Phoenix, Alby, Wallet of Satoshi, etc.

Lightning Address
cruxcoach@npub.cash

Any amount welcome, even a few sats. Zaps via Nostr also land at this address.

Contribute on Codeberg

Money isn't the only thing that helps. The cheapest contribution is feedback:

  • File bug reports or feature requests on the issue tracker
  • Send a pull request — see CONTRIBUTING.md for setup
  • Help with translations beyond English and German
  • Find a security issue? SECURITY.md describes responsible disclosure

Spread the word

If you climb at a gym with a Kilter Board and the official app keeps frustrating you, telling other climbers about CruxCoach is genuinely the highest-leverage thing you can do. The project grows by word of mouth — there's no marketing budget.

What we don't take

No subscriptions, no premium tier, no recurring billing, no "Pro" version. The whole project is GPL-3.0 and stays that way. If we ever set up a structured way to fund the project (OpenCollective, Geyser, etc.), it'll be disclosed here first — never via stealth changes.
