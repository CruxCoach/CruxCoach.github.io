About this map

Every dot is a climbing-board installation that someone published as "listed" in the upstream feed. Most are commercial climbing gyms; some are home setups whose owner chose to share the location. Coverage is best for Kilter and MoonBoard; smaller systems (Touchstone, Aurora, So iLL) only show the few publicly-listed walls.

Data sources

  • hangtime-climbing-boards — daily-updated GeoJSON dataset of Aurora-board installations (Kilter / Tension / Grasshopper / Decoy / So iLL / Touchstone / Aurora) plus MoonBoard and 12climb, released into the public domain.
  • Map tiles by OpenStreetMap contributors. Privacy implications on the privacy page.

Refresh cadence

We re-pull the upstream dataset periodically and commit the snapshot. Build timestamp + per-board counts: boards.meta.json.
